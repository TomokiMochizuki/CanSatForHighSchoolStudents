# IMU (9軸センサ) の単体テストコード (Pololu MinIMU-9 v5)
## 概要
IMU (9軸センサ) の単体テストコード  
9軸センサとは，以下のセンサがそれぞれ3軸あることによる

+ 加速度
+ 角加速度
+ 地磁気


## 機器詳細
